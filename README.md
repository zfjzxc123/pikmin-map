🌱 Pikmin Bloom 飾品地圖共筆系統

歡迎來到 Pikmin Bloom 飾品地圖！這是一個由玩家社群共同維護的地圖工具，旨在幫助大家更輕鬆地找到特定種類的飾品皮克敏（Decor Pikmin）。

無論你是想找稀有的「動物園 🦁」、「美術館 🎨」還是「漢堡店 🍔」，這個地圖都能讓你一目瞭然！

✨ 主要功能

🗺️ 真實地圖整合：使用 OpenStreetMap，清晰顯示街道與地點資訊。

🔍 強大篩選功能：

多選過濾：左側選單支援多選，想同時找「咖啡廳」與「甜點店」也沒問題。

智慧跳轉：點選左側分類時，地圖會自動飛到離你最近的該類別地點（需開啟定位）。

📍 輕鬆新增與編輯：

直覺操作：點擊地圖任意處即可新增。

智慧填入：選擇飾品類別後，名稱會自動預設為該類別名稱，省去打字麻煩。

座標支援：支援直接貼上 Google Maps 座標（例如 25.04, 121.53）。

資料修正：發現資訊有誤？點擊地點即可進行編輯或刪除。

🚀 導航輔助：

一鍵定位：右下角定位按鈕，快速鎖定目前位置。

複製地址/導航：點擊地圖上的圖示，可快速複製地址並開啟 Google Maps 進行導航。

☁️ 雲端即時同步：所有資料皆儲存於雲端資料庫，任何人新增的地點，其他人重新整理後都能立刻看到。

📖 如何使用

1. 瀏覽地圖

打開網頁後，地圖會顯示台灣地區。

點擊右下角的 「🎯 定位」 按鈕，允許瀏覽器存取位置，地圖將移動到您所在的地方。

使用 左側選單（手機版點左上角選單鈕）點擊您感興趣的飾品類別。地圖上只會保留您選中的類別圖示。

2. 新增地點

想要分享您發現的寶地嗎？

點擊右上角的 「+ 新增地點」 按鈕。

方式 A (點擊)：直接在地圖上點擊該地點的位置。

方式 B (座標)：點擊上方橫幅的「手動輸入座標」，貼上經緯度。

在表單中選擇 「飾品類別」（名稱會自動帶入，您也可以修改成更詳細的店名，例如：星巴克 台北101店）。

按下 「確認新增」。

3. 編輯與刪除

點擊地圖上既有的圖示。

在彈出的視窗中，您可以選擇 「編輯」 修改資訊，或是 「刪除」 錯誤的點位。

4. 導航前往

點擊地圖上的圖示。

點擊 「📋 複製地址」。

系統會詢問是否要開啟 Google Maps，確認後即可開始導航。

⚠️ 注意事項

本服務為共筆性質，請大家共同維護資料的準確性。

請勿惡意刪除他人標記的正確地點。

新增地點時，建議備註該地點的偵測範圍或特殊條件（例如：只有站在門口才偵測得到）。
